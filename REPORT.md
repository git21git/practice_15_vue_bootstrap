# Отчёт по практической работе 15

## 1. Название практической работы

Практическая работа 15. JS-фреймворки + CSS-фреймворки.

## 2. Цель работы

Цель работы: изучить совместное использование JavaScript-фреймворка Vue 3 и CSS-фреймворка Bootstrap 5, создать адаптивное клиентское приложение и подготовить его к публикации на GitHub Pages.

## 3. Используемые технологии

- Vite;
- Vue 3;
- JavaScript;
- Bootstrap 5;
- HTML;
- CSS;
- localStorage;
- Git;
- GitHub Pages.

## 4. Почему выбрано сочетание Vue 3 + Bootstrap

Vue 3 удобен для создания интерактивных интерфейсов: он позволяет использовать реактивные данные, вычисляемые значения, двустороннее связывание форм и обработку событий. Bootstrap 5 ускоряет разработку внешнего вида, потому что предоставляет готовые классы для адаптивной сетки, форм, кнопок, карточек, навигации, уведомлений и бейджей. Такое сочетание помогает быстро создать рабочее приложение с аккуратным адаптивным интерфейсом без большого количества собственного CSS.

## 5. Описание реализованного приложения

Реализовано приложение «Моя коллекция книг». Пользователь может вести личную библиотеку: добавлять книги, указывать автора, описание и ссылку на обложку, просматривать коллекцию в карточках, искать нужные книги, сортировать список и применять фильтры. Коллекция сохраняется в браузере и восстанавливается после обновления страницы.

## 6. Реализованный функционал

- добавление книги через форму;
- поля книги: название, автор, описание, ссылка на фото обложки;
- отображение книг в Bootstrap-карточках;
- удаление книги;
- поиск по названию или автору;
- сортировка по названию в прямом и обратном порядке;
- фильтр по наличию описания;
- фильтр по наличию фото обложки;
- адаптивная сетка карточек через Bootstrap `row` и `col`;
- сохранение коллекции в `localStorage`;
- восстановление коллекции после обновления страницы.

## 7. Использованные Bootstrap-классы и для чего они применены

- `container` — ограничение ширины и выравнивание основного содержимого;
- `row` — создание строк адаптивной сетки;
- `col` — базовая колонка Bootstrap;
- `col-12`, `col-md-6`, `col-xl-4` — настройка ширины колонок на разных экранах;
- `card` — оформление карточек книг;
- `card-body` — внутренняя область карточки;
- `card-title` — заголовок карточки с названием книги;
- `btn` — базовый стиль кнопок;
- `btn-primary` — кнопка добавления книги;
- `btn-danger` — кнопка удаления книги;
- `form-control` — оформление полей ввода, textarea и select;
- `input-group` — объединение подписи и поля поиска;
- `alert` — сообщения формы и пустого состояния списка;
- `badge` — счётчики и статусы карточек;
- `navbar` — верхняя панель приложения.

## 8. Использованные возможности Vue 3

- `<script setup>` — компактный синтаксис компонента;
- `ref` — хранение реактивных данных формы, списка книг, поиска, фильтра и сортировки;
- `computed` — получение отфильтрованного и отсортированного списка книг;
- `watch` — сохранение коллекции в `localStorage` при изменении списка;
- `v-model` — двустороннее связывание полей формы и элементов управления;
- `v-for` — вывод списка карточек книг;
- `v-if` — условное отображение обложки, описания, сообщений и пустого состояния;
- `@submit.prevent` — обработка отправки формы добавления книги;
- `@click` — обработка удаления книги.

## 10. Работа с localStorage

Для сохранения коллекции используется ключ `practice-15-book-collection`. При запуске приложение пытается прочитать данные из `localStorage`. Если сохранённых данных нет или они повреждены, используются начальные демонстрационные книги. При каждом изменении массива книг срабатывает `watch`, который записывает актуальное состояние коллекции в `localStorage`. Благодаря этому после обновления страницы добавленные книги остаются в приложении.

## 11. Инструкция по запуску проекта локально

1. Установить зависимости:

```bash
npm install
```

2. Запустить сервер разработки:

```bash
npm run dev
```

3. Открыть адрес, который будет показан в терминале.

## 12. Инструкция по сборке проекта

Для создания production-сборки нужно выполнить команду:

```bash
npm run build
```

После выполнения команды готовые файлы появятся в папке `dist`.

## 13. Инструкция по публикации на GitHub Pages

1. Создать или открыть репозиторий:

```text
https://github.com/git21git/practice_15_vue_bootstrap
```

2. Подключить удалённый репозиторий:

```bash
git remote add origin https://github.com/git21git/practice_15_vue_bootstrap.git
```

3. Подготовить ветку `main` и отправить код:

```bash
git branch -M main
git push -u origin main
```

4. Выполнить публикацию:

```bash
npm run deploy
```

5. Включить GitHub Pages: GitHub → репозиторий `practice_15_vue_bootstrap` → Settings → Pages → Source: Deploy from a branch → Branch: `gh-pages` → Folder: `/root` → Save.
