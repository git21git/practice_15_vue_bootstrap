# Моя коллекция книг

Практическая работа 15 по теме «JS-фреймворки + CSS-фреймворки». Проект реализован на Vite + Vue 3 без TypeScript с использованием Bootstrap 5. Автор: Чудинова Мария.

Репозиторий: https://github.com/git21git/practice_15_vue_bootstrap

GitHub Pages: https://git21git.github.io/practice_15_vue_bootstrap/

## Краткое описание

«Моя коллекция книг» — адаптивное приложение для ведения личной библиотеки. Пользователь добавляет книги через форму, видит их в Bootstrap-карточках, ищет по названию или автору, сортирует список и фильтрует коллекцию по наличию описания или фото обложки. Данные сохраняются в `localStorage` и восстанавливаются после обновления страницы.

## Функции

- добавление книги с названием, автором, описанием и ссылкой на фото обложки;
- отображение коллекции в Bootstrap-карточках;
- удаление книги;
- поиск по названию или автору;
- сортировка по названию;
- фильтр по наличию описания;
- фильтр по наличию фото обложки;
- адаптивная сетка карточек через `row` и `col`;
- сохранение и восстановление коллекции через `localStorage`.

## Используемые технологии

## Почему Vue 3 + Bootstrap

Vue 3 выбран для реактивной логики интерфейса: формы, списки, вычисляемые данные, фильтрация и обработка событий удобно реализуются средствами фреймворка. Bootstrap 5 выбран как CSS-фреймворк, который быстро даёт готовую адаптивную сетку, карточки, формы, кнопки, навигацию, бейджи и уведомления. Вместе они позволяют сделать понятное и аккуратное приложение без сложного собственного CSS.
